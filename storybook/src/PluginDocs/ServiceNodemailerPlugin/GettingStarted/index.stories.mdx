# Getting Started

##

<br />

The **Nodemailer** plugin sets up a **Nodemailer configuration** for sending emails in your application. With this plugin, you can easily integrate email sending functionality into your project. The plugin provides a simple and efficient way to configure your email settings and start sending emails right away.
